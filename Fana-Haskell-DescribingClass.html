Fana.Haskell.DescribingClass

Description

Suppose that type t has a class that describes it totally. By 'describing totally' i mean that any value of any type that also has that class can be converted to t. This module provides a class for such types. More accuretely multiple classes, one for each n in 0..4 where n is the number of parameters in the kind of t.
