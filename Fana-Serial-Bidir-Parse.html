Parser.

  • "p" is id of position in input stream.
  • "v" is the type of parsed value.

The state monad covers the input stream. The Either monad handles parse errors.

The state monad's structure is (s -> m (v, s)), where m is the either monad. Thus failed parsings do not alter the input stream.
