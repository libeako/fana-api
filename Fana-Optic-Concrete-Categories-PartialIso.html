test_piso :: Eq h1 => (Iso' l1 l2, Iso' h1 h2) -> [l2] -> [h1] -> PartialIso e l1 l2 h1 h2 -> Bool #

Goes down then up along the given partial isomorphism and examines whether it ends back where started.

This function tests both success and fail cases. Fail is expected on the given low level data. Success is expected on the given high level data.
