data FlatSumCase c c' s s' #

One case of a flat sum.

  • s is the [conceptual] sum type.

Contains 2 serialization layers. The lower one works on the stream and connects it to an intermediate value level, which technically is the type of the value that this sum case serializes. The higher one connects the intermediate to the sum type level. I use the name m for the type of the intermediate level.
